📊 YouTube Global Trends: Inteligência de Dados para Estratégia de Ads
📝 Sobre o Projeto
Este projeto analisa o ecossistema de tendências globais do YouTube para transformar grandes volumes de dados em insights acionáveis. O foco principal é apoiar o planeamento de anúncios e a criação de conteúdo, identificando onde a audiência está mais recetiva e quais os padrões de consumo que maximizam o ROI (Retorno sobre Investimento).

Utilizando uma abordagem data-driven, o projeto permite a transição de um marketing reativo para uma estratégia baseada em padrões de comportamento e retenção.

🚀 Principais Insights de Negócio
1. Retenção e Ciclo de Vida do Conteúdo
Liderança Global: As categorias de Entertainment (Entretenimento) e Music (Música) são as mais frequentes na secção de tendências mundialmente.

Frequência de Retenção: Identificou-se que vídeos de entretenimento possuem a maior taxa de permanência na lista de tendências, mantendo o engajamento por vários dias consecutivos.

2. Segmentação Regional e Audiência
Mercado Indiano: Apresenta o maior volume absoluto de vídeos em tendência, com uma audiência massiva e concentrada em entretenimento.

Diversidade nos EUA: O mercado norte-americano demonstra uma maior segmentação de interesses, com categorias como How-to & Style, Comedy e News & Politics a performarem acima da média global.

🛠️ Tecnologias e Metodologia
Data Visualization: Tableau Public (Dashboards interativos).

Storytelling: Gamma (Apresentação executiva).

Processamento: Análise de histórico de tendências e distribuição transregional.

📈 Planeamento e Próximos Passos
Monitorização Dinâmica: Implementação de rotina de acompanhamento de categorias em ascensão.

Integração com Ads: Alinhamento dos dados com o planeamento de anúncios para otimizar o alcance orgânico.
